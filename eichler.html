The Eichler Homepage

< link to project >

about

The Eichler Homepage, is visual storytelling/text-based art project that creates an interactive experience for the user as they learn about the mythical reality of the Mid-Century Modern Eichler home. The project exists between the art and digital making space, not quite a game, but not quite a story, either.

The project was a culmination of research, journal entries, and visual interpretations of a term-long project in my art class, Obsessive Investigation led by my professor Deb Sokolow, who uses a similar process when creating her own work.

gameplay

The user is carried through pages of text and story that explain my own understanding and relationship with Eichlers, as well as content attempting to understand the sociological/economic implications of the upper-middle class.

code

The project utilized HTML and CSS to create a simple storytelling experience that allowed players to go through the project in succession. I also used a CSS grid to be able to create a page of links where users could explore an interactive flowchart.

visual components

The visual and textual components really intersected in the project as I often displayed handwritten and draw work in tandum, and utilized post-it notes, and calligraphy sheets to background different textual themes/goals.

I utilized both physical and digital drawing to enhance the visual experience for the user. This way I could include things like simple animation work and more refined digital drawing, alongside rougher sketches and writing. This really emphasized the changes in voice from page to page.
